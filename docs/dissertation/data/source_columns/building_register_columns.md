# 건축물대장 원천 컬럼 정의

## 원천파일 형식 메모
- 대용량 내려받기 원천 .txt는 헤더 없이 데이터 행부터 시작함.
- 필드 구분자는 | 이며, 컬럼명 매칭은 컬럼 순서 기준으로 수행해야 함.

## 연구에 우선 필요한 핵심 컬럼(본 연구 기준)
- 식별/공간: `관리_건축물대장_PK`, `시군구_코드`, `법정동_코드`, `대지_위치`
- 일정(핵심 결과변수): `허가_일`, `착공_일`, `사용승인_일`
- 건물 특성 통제: `주_용도_코드`, `주_용도_코드_명`, `연면적(㎡)`, `건축_면적(㎡)`, `세대_수(세대)`, `호_수(호)`, `용적_률(%)`, `건폐_율(%)`
