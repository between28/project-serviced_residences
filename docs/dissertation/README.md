# Dissertation Documentation

This folder is the canonical documentation source for the University of Cambridge
MPhil in Planning, Growth and Regeneration dissertation project.

## Structure
- `proposal/`: proposal and detailed research design text
- `design/`: hypotheses and identification strategy
- `data/`: variable definitions and source-field map
- `policy/`: policy event timelines and coding rules
- `admin/`: programme context, governance, and delivery standards

## Working Rule
- Keep substantive research content in this folder.
- Keep root-level markdown files as lightweight entry points only.
